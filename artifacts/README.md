# Artifacts

This folder stores outputs generated by the pipeline and supporting materials for reproducibility.

Place final exports here before submission.

Files (placeholders)

- `prisma_counts.json` — PRISMA flow counts and reasons.
- `records_deduped.csv` — Deduplicated records after initial search.
- `search_results/` — Raw exports from each connector (JSON files).
- `excluded_papers_82.csv` — List of excluded full-text papers with reasons.
- `screening_decisions.csv` — Screening log for title/abstract and full-text decisions.
